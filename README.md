# Clustering-ML-of-Person-of-interest-in-Epstein-Files
This repo is a source code of a machine learning model from a person name whos mentioned in Epstein files


This ML model is clustering from the person who have connection with Jeffrey Epstein and its separating them into 4 class
- Class 1 periferal guest
- class 2 knew Jeffrey Epstein 
- class 3 regular passenger
- Class 4 Jeffrey Epstein himself

this model using 3 algorithm with accuracy higher than 90% and have 3 splitting ratio
70% train, 30% test
80% train, 20% test
90% train, 10% test
